# Qarin — Logo Rules

**Version:** 0.1.0 · 2026-06-22
**Status:** PLACEHOLDER — Arabic calligraphy mark pending

---

## The Mark

Qarin — `قرين` — in Arabic calligraphy. Qaf → Ra → Ya → Noon.

Connected. Flowing. A single continuous stroke where possible.

## Lockups

| Variant | File | Use |
|---|---|---|
| Primary | `primary.svg` | Default. Wordmark + icon or wordmark only |
| Icon | `icon.svg` | Favicon, app icon, small spaces |

## Clear Space

Minimum clear space on all sides: **1× the height of the Qaf** (the tallest letter).

For the icon-only mark: **1× the mark height**.

## Minimum Size

- Primary lockup: 120px wide (print), 36px height (screen)
- Icon only: 16px (favicon), 32px (app)

## Don't

- Stretch, skew, or rotate
- Change the color to anything outside the palette
- Add drop shadows or effects
- Place on busy backgrounds
- Outline it
- Use a different Arabic font and call it the logo

## Backgrounds

- Preferred: white or `#F7F6FA`
- On dark: use the `--sa-text-inverse` variant if provided
- Never on a color that competes with the lavender palette

## Temporary

Until the calligraphy mark is delivered:
- The fleet dashboard uses "QARIN" in Plus Jakarta Sans as a placeholder wordmark
- No icon is used

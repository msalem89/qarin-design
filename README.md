# Qarin Design System

The design language for Qarin — an AI companion platform by Samaana Solutions. Clean, restrained, high-trust.

## Quickstart

```html
<link rel="stylesheet" href="identity/tokens.css">
<link rel="stylesheet" href="components/tables.css">
```

## Docs

Live at: https://msalem89.github.io/qarin-design/site/

## Structure

```
identity/    ← Tokens, palette, typography, logo rules
components/  ← One CSS file per component
pages/       ← Reference implementations
site/        ← Design system documentation site
catalogs/    ← Product catalogs (company IP)
plans/       ← Finalized design decisions
archive/     ← Superseded work
```

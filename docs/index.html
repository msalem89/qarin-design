Usage

Two files. Import and go.

<!-- In your <head> --> <link rel="stylesheet" href="identity/tokens.css"> <link rel="stylesheet" href="components/tables.css">

Never hardcode values. Always consume tokens via var(--sa-*). Add new components in components/ — one file per component. See README.md for the full rules.
